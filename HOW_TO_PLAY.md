# Need for Drift — How to Play

Need for Drift is a drift-racing game: pick your car, drift around the track, and lap it as fast as you can!

## Install & launch

### Windows

1. Unzip `NeedForDrift-Windows.zip`.
2. Open the extracted folder.
3. Run `Need for Drift.exe`.

> Keep every file together: the `.exe` needs the `Need for Drift_Data` folder right next to it.

### macOS

1. Unzip `NeedForDrift-macOS.zip`.
2. **Right-click** `Need for Drift.app` → **Open** (the app isn't signed, so macOS asks for confirmation the first time).

> If macOS still refuses to open it, run this once in the Terminal:
> ```bash
> xattr -dr com.apple.quarantine "Need for Drift.app"
> ```

## Controls

| Action | Key |
|---|---|
| Accelerate | `W` |
| Brake / Reverse | `S` |
| Steer left | `A` |
| Steer right | `D` |
| Drift (handbrake) | `Space` |
| Pause menu | `Esc` |

> On an AZERTY keyboard, `W` `A` `S` `D` are physically the `Z` `Q` `S` `D` keys.
> In the intro, press `Left Shift` to reset the bus if it flips over.

## How it plays

1. **Intro** — you start out driving the bus.
2. **Select your car** — pick one of 5 vehicles, each with its own speed and handling.
3. **Race** — drift through the track and chase your best lap time.

## Goal

Master the drift, carry your speed through the corners, and finish the track as fast as possible.
